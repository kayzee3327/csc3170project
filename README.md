# CSC3170 Project

To run this project,

1. Pull from github
   ```bash
   git clone https://github.com/kayzee3327/csc3170project.git
   cd csc3170project
   ```
2. Run docker compose (make sure docker engine is running)
   ```bash
   docker compose up
   ```

To stop the server,

Type `ctrl-C` in the log terminal. Or use command `docker compose down` in another terminal
